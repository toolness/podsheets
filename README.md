This web app makes it easy to publish RSS podcast feeds from Google
Spreadsheets.

## Quick Start

```
$ git clone https://github.com/toolness/podsheets.git
$ cd podsheets
$ npm install
$ DEBUG= node podsheets.js
```

Then visit http://localhost:3000 for instructions.
